# ARO Website

Website for Artists Reaching Out, a Columbia University student club that brings visual arts education to youth in Morningside Heights and Harlem.

## Files

- `index.html` - Home page
- `volunteers.html` - Volunteer information
- `resources.html` - Resources page
- `styles.css` - Visual styling
- `images/` - Logo and photos

## How to Edit

1. Open any `.html` file in a text editor (TextEdit, Notepad, or VS Code)
2. Find the text you want to change
3. Edit the text between the `>` and `<` symbols
4. Save the file

Each section in the HTML files has comments (gray text starting with `<!--`) explaining what to edit.

### Common Updates

**Change coordinator info:** Edit the Leadership section in `index.html`

**Add a partner:** Copy an existing `<li class="partner">...</li>` line in the Partners section and change the name

**Update contact email:** Search for the old email and replace it with the new one

**Replace an image:** Put the new image in the `images/` folder with the same filename
